## How to Run This Project

### **1. Clone the Repository**
```bash
git clone <repository-url>
cd <repository-folder>
```

### **2. Install Dependencies**
```bash
npm install
```

### **3. Run the Project**
```bash
npm run dev
```

The project will be available at: [http://localhost:3000](http://localhost:3000)
